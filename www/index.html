Meet Locker is an application that allows users to store and retrieve files.

On the first screen you enter the data to store and a password, then you get a key to the locker that stores the data using your password.

On the next screen you enter the key and your password, and you get your data back.

This version of the application is a proof-of-concept demostrating the code involved with doing something like this.

The application is distributed via source code at GitHub

It is a portable ASP.NET Core Web API Kestrel program
